# Contributing to Coconut App

Thanks for your interest in contributing to the Coconut mobile app.

## Getting started

1. **Fork and clone** the repo.
2. **Install dependencies:** `npm install`
3. **Start the app:** `npx expo start`
4. **Create a branch** for your change: `git checkout -b your-feature`

## Development

- **Expo Go:** Use the QR code for quick iteration on Home & Shared flows.
- **Dev build (Tap to Pay):** Run `npx expo run:ios` or `npx expo run:android` on a physical device.

## Submitting changes

1. Keep commits focused and messages clear.
2. Open a pull request against `main` with a short description of the change.
3. Ensure the app runs with `npx expo start` and, if relevant, a dev build.

## Code style

- TypeScript throughout. Follow existing patterns in the codebase.
- Use Expo Router file-based routing under `app/`.

## Questions

Open an issue for bugs, feature ideas, or questions.
