# Code of Conduct

## Our standards

We expect everyone involved in Coconut App to:

- Be respectful and inclusive.
- Welcome constructive feedback and differing viewpoints.
- Focus on what’s best for the community and the project.
- Not harass, troll, or use discriminatory language.

## Scope

This code of conduct applies to project spaces (repos, issues, PRs, discussions) and to anyone representing the project in public.

## Enforcement

Unacceptable behavior may be reported by opening an issue or contacting the maintainers. We will treat reports confidentially and take appropriate action.

## Attribution

This code of conduct is adapted from common open-source community standards.
